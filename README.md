# CatTalk 2.0_android
--------
## 简介
本Demo使用环信SDK创建的完整的类微信的聊天APP。
功能包括：注册新用户，用户登录，添加好友，单聊，群聊，发送文字，表情，语音，图片，地理位置等消息，以及实时音视频通话等。
